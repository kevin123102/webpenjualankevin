Deskripsi Website Toko Online

Fitur Utama

  1. Halaman Login:
    • User harus login menggunakan form yang disediakan.
    • Jika login berhasil, user akan diarahkan ke halaman beranda.
  2. Halaman Beranda dan Produk:
    • User dapat melihat produk yang tersedia di halaman produk.
    • Tombol "Beli Sekarang" memungkinkan user untuk menambahkan produk ke keranjang belanja.
  3. Fitur Keranjang Belanja:
    • Keranjang berisi daftar produk yang dipilih dan total harga.
    • User dapat melanjutkan ke halaman pembayaran dengan menekan tombol "Checkout".
  4. Halaman Pembayaran dan Riwayat:
    • Setelah checkout, user akan diarahkan ke halaman pembayaran.
    • User dapat melihat riwayat pembelian di halaman "Riwayat".
